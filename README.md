# ENDE-UD04-A2

## Descripción
Repositorio creado para la práctica **UD04 – Actividad 2** de la asignatura **Entornos de Desarrollo**.  
El objetivo es aprender a utilizar **Git** y trabajar con repositorios remotos en plataformas como GitHub.

## Contenido
El repositorio incluye el proyecto **Figuras** en Java, que contiene varias clases relacionadas con figuras geométricas. Estas clases se utilizan para practicar el control de versiones.

## Actividades realizadas
Durante la práctica se han realizado las siguientes tareas:

- Creación de un repositorio remoto.
- Clonación del repositorio en local.
- Añadir archivos al *staging area*.
- Realización de varios **commit**.
- Consulta del historial con `git log`.
- Refactorización de clases.
- Creación y uso de ramas (`master` y `develop`).
- Subida de cambios al repositorio remoto con `push`.
- Trabajo con repositorios de otros compañeros.
- Uso de **Pull Request** para integrar cambios entre ramas.

## Tecnologías utilizadas
- Java
- Git
- GitHub
